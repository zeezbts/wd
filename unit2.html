CSS Styling (Background, Text, Fonts)

Background Properties

  • background-color: Sets solid colors using color names, HEX, RGB, or HSL.
  • background-image: Embeds an image file using url('image.jpg').
  • background-repeat: Prevents or allows tiling (repeat, no-repeat, repeat-x).
  • background-position: Aligns background graphic locations (center, top right).

Text Formatting

  • color: Controls foreground text colors.
  • text-align: Sets horizontal alignment (left, right, center, justify).
  • text-decoration: Attaches or removes lines (none, underline, line-through).
  • text-transform: Forces case conversions (uppercase, lowercase, capitalize).
  • letter-spacing / word-spacing: Adjusts text spacing intervals.

Controlling Fonts

  • font-family: Establishes font choice with fallback stacks.
  • font-size: Defines scale (px, em, rem, %).
  • font-style: Toggles slant variations (normal, italic).
  • font-weight: Adjusts structural boldness (normal, bold, 100-900).
