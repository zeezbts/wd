JavaScript Core Syntax Reference

1. Introduction to JavaScript
  • Definition: High-level, single-threaded, dynamic interpreted browser scripting language.
  • Role: Manages behavioral actions and user runtime interactivity layers on client screens.
  • Execution: Code compiles and executes directly inside native client engines (e.g., V8).
2. Variables & Scoping Bounds
  • var: Legacy variable scope definition keyword. Function-scoped, prone to hoisting issues.
  • let: Modern block-scoped variable declaration. Values can be safely reassigned over time.
  • const: Block-scoped read-only constant declaration. Reassignment is strictly prohibited.
let itemTotal = 450; const conversionRate = 0.82;
3. Functional Declarations
  • Mechanics: Packaged blocks of logic that accept parameters and run only when explicitly invoked.
  • Return Payload: Employs explicit return commands to pass calculated data payloads back to callers.
function evaluateTax(amount) { return amount * 0.15; }
4. Conditionals & Repetition Loops
  • Branching: Controls logical execution flows using if / else if / else structural blocks.
  • Loops: Runs a block of code repeatedly using for or while loops as long as a specified condition remains true.
// Repetitive execution loop template for (let step = 0; step < 5; step++) { console.log("Current Step: " + step); }
