JavaScript Core Syntax Reference

1. Introduction to JavaScript
  • Definition: High-level, single-threaded, dynamic interpreted browser scripting language.
  • Role: Manages behavioral actions and user runtime interactivity layers on client screens.
  • Execution: Code compiles and executes directly inside native client engines (e.g., V8).
2. Variables & Scoping Bounds
  • var: Legacy variable scope definition keyword. Function-scoped, prone to hoisting issues.
  • let: Modern block-scoped variable declaration. Values can be safely reassigned over time.
  • const: Block-scoped read-only constant declaration. Reassignment is strictly prohibited.
let itemTotal = 450; const conversionRate = 0.82;
3. Functional Declarations
  • Mechanics: Packaged blocks of logic that accept parameters and run only when explicitly invoked.
  • Return Payload: Employs explicit return commands to pass calculated data payloads back to callers.
function evaluateTax(amount) { return amount * 0.15; }
4. Conditionals & Repetition Loops

1. Conditional Statements

JavaScript Conditionals

  • if Statement: A basic decision structure that executes a block of code only if its specified condition evaluates to true.

    Syntax:

    if (condition) {
  // Code runs if condition is true
}

    Example:

    let age = 20;
if (age >= 18) {
  console.log("Allowed to vote");
}
  • else Statement: Paired with an if statement to provide a fallback code block that runs only when the initial condition is false.

    Syntax:

    if (condition) {
  // Runs if condition is true
} else {
  // Runs if condition is false
}

    Example:

    let score = 45;
if (score >= 50) {
  console.log("Passed");
} else {
  console.log("Failed");
}
  • else if Statement: Evaluates a new condition if the preceding condition evaluates to false, allowing multiple sequential checks.

    Syntax:

    if (condition1) {
  // Runs if condition1 is true
} else if (condition2) {
  // Runs if condition1 is false and condition2 is true
}

    Example:

    let speed = 75;
if (speed > 80) {
  console.log("Too fast");
} else if (speed > 60) {
  console.log("Normal speed");
}
  • if-else-if Ladder: A chained series of else if statements used to evaluate multiple mutually exclusive conditions in a top-down order.

    Syntax:

    if (condition1) {
  // Code block 1
} else if (condition2) {
  // Code block 2
} else if (condition3) {
  // Code block 3
} else {
  // Default code block
}

    Example:

    let marks = 85;
if (marks >= 90) {
  console.log("Grade A");
} else if (marks >= 80) {
  console.log("Grade B");
} else if (marks >= 70) {
  console.log("Grade C");
} else {
  console.log("Grade D");
}
  • Nested if Statement: An if statement placed inside another if statement, used to test secondary conditions only after a primary condition is met.

    Syntax:

    if (outerCondition) {
  if (innerCondition) {
    // Runs only if both conditions are true
  }
}

    Example:

    let hasAccount = true;
let balance = 150;
if (hasAccount) {
  if (balance > 100) {
    console.log("Transaction approved");
  }
}
  • switch Statement: A multi-way branch structure that compares a single expression value against predefined case labels using strict equality.

    Syntax:

    switch (expression) {
  case value1:
    // Code block
    break;
  case value2:
    // Code block
    break;
  default:
    // Fallback code block
}

    Example:

    let day = 2;
switch (day) {
  case 1:
    console.log("Monday");
    break;
  case 2:
    console.log("Tuesday");
    break;
  default:
    console.log("Weekend or invalid day");
}

2. Repetition Loops

Repetition Loops Definitions, Syntax, and Examples

  • for Loop: A counting loop that packs counter setup, condition check, and step modification into a single control line.

    Syntax:

    for (initialization; condition; final-expression) {
  // Code block to repeat
}

    Example:

    for (let i = 1; i <= 3; i++) {
  console.log("Count: " + i);
}
  • while Loop: An entry-controlled structure that repeats a statement block as long as its condition remains true.

    Syntax:

    while (condition) {
  // Code block
}

    Example:

    let energy = 2;
while (energy > 0) {
  console.log("Running...");
  energy--;
}
  • do...while Loop: An exit-controlled loop that processes the statement block once before evaluating the condition.

    Syntax:

    do {
  // Code block
} while (condition);

    Example:

    let attempts = 0;
do {
  console.log("Checking...");
} while (attempts > 0);
  • for...of Loop: A collection iterator that extracts values directly from iterable sequences like arrays or strings.

    Syntax:

    for (const value of iterable) {
  // Code block
}

    Example:

    const colors = ["red", "blue"];
for (const color of colors) {
  console.log(color);
}
  • for...in Loop: A key investigator loop that traverses the enumerable string property keys of an object.

    Syntax:

    for (const key in object) {
  // Code block
}

    Example:

    const user = { name: "Alex", age: 25 };
for (const key in user) {
  console.log(key + ": " + user[key]);
}
