Introducing Cascading Style Sheets & JavaScript

CSS Overview

CSS is a stylesheet language that describes the presentation of an HTML (or XML) document.

CSS describes how elements must be rendered on screen, on paper, or in other media.

This tutorial will teach you CSS from basic to advanced.

What is CSS?

  • CSS stands for Cascading Style Sheets.
  • CSS describes how HTML elements are to be displayed on screen, paper, or in other media.
  • CSS saves a lot of work. It can control the layout of multiple web pages all at once.
  • External stylesheets are stored in CSS files.

CSS Architecture & Implementation

Why Use CSS?

CSS is used to define styles for your web pages, including the design, layout and variations in display for different devices and screen sizes.

Syntax Example

Below is an example of a CSS rule-set styling an HTML heading element:

h1 {
  color: blue;
  font-size: 20px;
}

The selector points to the HTML element you want to style.

The declaration block contains one or more declarations separated by semicolons.

Each declaration includes a CSS property name and a value, separated by a colon.

A CSS declaration always ends with a semicolon, and declaration blocks are surrounded by curly braces.

Three Ways to Insert CSS

Styling can be added to HTML elements in 3 ways:

  • Inline: Using a style attribute inside HTML elements.
  • Internal: Using a <style> element inside the HTML <head> section.
  • External: Using one or more external CSS files.

The most common way to add styling is to keep the styles in separate CSS files.

Three Ways to Insert CSS

Inline Styling (Inline CSS)

Inline styling is used to apply a unique style to a single HTML element. It uses the style attribute directly on the tag.

This example changes the text color of the <h1> element to blue:
HTML Example
<h1 style="color:blue;">This is a Blue Heading</h1>

Internal Styling (Internal CSS)

Internal styling is used to define a style for one single HTML page. It is defined within the <head> section of an HTML page inside a <style> element.

HTML Page Content
<!DOCTYPE html>
<html>
<head>
<style>
  body { background-color: lightgrey; }
  h1 { color: blue; }
  p { color: green; }
</style>
</head>
<body>
  <h1>This is a heading</h1>
  <p>This is a paragraph.</p>
</body>
</html>

External Styling (External CSS)

An external style sheet is used to define the style for many pages. With an external style sheet, you can change the look of an entire website by changing just one file!

To use it, add a link to the external file inside the <head> section of your HTML page.

The file should not contain any HTML tags and must be saved with a .css extension.
index.html
<!DOCTYPE html>
<html>
<head>
  <link rel="stylesheet" href="styles.css">
</head>
<body>
  <h1>This is a heading</h1>
  <p>This is a paragraph.</p>
</body>
</html>
styles.css
body {
  background-color: lightgrey;
}
h1 {
  color: blue;
}
p {
  color: green;
}

Grouping Styles

  • Grouping Selectors

    Combine multiple elements sharing the same styles by separating them with commas.

    h1, h2, p {
  text-align: center;
  color: gray;
}

  • Grouping Declarations

    Combine multiple style properties for a single element inside one set of curly braces.

    pre {
  background-color: lightgray;
  padding: 10px;
  border: 1px solid black;
}

Extensible Markup Language (XML)

  • Purpose: XML purely stores and structures data using completely custom, user-defined tags. It has no built-in styling.
  • CSS Link: To display XML data visually in a browser, you must link an external CSS file at the very top of the document. 
    <?xml-stylesheet type="text/css" href="xml-styles.css"?>
